===========

Neighbourhood Map
===========

## To install all dependencies
```
npm install
```
## To run
```
npm run dev
```
Files will be served on localhost:8080

## Features
* Responsive
* Users may also add a location by clicking the '+' sign in the infowindow
* Offline storage with localStorage
